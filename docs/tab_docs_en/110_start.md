# Getting started

:::note Prepare

âœ…Required **Flutter** to install.

:::

<br />

# 1. Install Comet

```
$ dart pub global activate comet_cli
```

<br />

# 2. Create new project

Please name your project like `my_site`.

```
$ comet create my_site
```

<br />

# 3. See your site on local browser

```
$ cd my_site
$ comet start
```
