# 始めましょう💫

:::note 確認

✅この作業には **Flutter** の事前インストールが必要です

:::

<br />

# 1. Comet をインストール

```
$ dart pub global activate comet_cli
```

<br />

# 2. 新規サイトを作成

作成されるフォルダに好きな名前をつけてください。例: `my_site`

```
$ comet create my_site
```

<br />

# 3. ローカルでサイトを確認

```
$ cd my_site
$ comet start
```

<br />
